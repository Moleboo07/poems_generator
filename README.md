# Projet Python : Créer un générateur de poèmes

Ce générateur de poèmes s'appuie sur les célèbres sagas que sont Harry Potter, Le Seigneur des Anneaux et Star Wars. L'utilisateur choisit sa saga favorie, un personnage qu'il affectionne ainsi que le type de poème souhaité et peut générer un poème qui répondra à ses critères.
